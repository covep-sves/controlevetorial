# Guia de Campo DCV — São Luís/MA

Aplicativo PWA desenvolvido para apoio às ações de Vigilância em Saúde, Controle Vetorial e Endemias da SEMUS São Luís.

## Objetivo
Oferecer apoio rápido à tomada de decisão em campo por:
- Agentes de Combate às Endemias (ACE);
- supervisores;
- profissionais da vigilância;
- equipes técnicas.

## Funcionalidades
- Decisão rápida em campo;
- Fluxos operacionais;
- Visita domiciliar;
- Algoritmo de aplicação de larvicida;
- Orientação sobre vetores e endemias;
- Fluxos para:
  - caramujo africano;
  - barbeiro;
  - focos de água parada;
  - insetos com possível interesse em saúde pública;
- Conteúdo educativo;
- Contatos institucionais.

## Características técnicas
- PWA instalável;
- Funciona offline após primeiro acesso;
- Sem login;
- Sem coleta de dados;
- Otimizado para celular.

## Estrutura principal
- index.html → interface principal;
- service-worker.js → cache/offline;
- manifest.json → instalação PWA;
- imagens PNG → ilustrações e identidade visual.
